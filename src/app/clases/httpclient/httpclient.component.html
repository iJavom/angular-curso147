Http Client


El http cliente es un modulo que pertenece a la liberia de angular/common/http y es la que se encarga de crear las peticiones http a los servidores. Es importante que la importemos en el modulo donde se haga uso.


Los metodos qeu podemos usar de la libreria son:


  • GET: Sirve para hacer solicitud de datos.
  • POST: Sirve para agregar datos.
  • PUT: Sirve para actualizar datos.
  • DELETE: Sirve para eliminar datos
