Http Client


El http cliente es un modulo que pertenece a la liberia de angular/common/http y es la que se encarga de crear las peticiones http a los servidores. Es importante que la importemos en el modulo donde se haga uso.


Los metodos qeu podemos usar de la libreria son:


  • GET: Sirve para hacer solicitud de datos. (Generalmente solo recibe un parametro que es el destino de la peticion)
  • POST: Sirve para agregar datos. (Recibe dos parametros que son el destino y el cuerpo)
  • PUT: Sirve para actualizar datos.(Recibe dos parametros que son el destino y el cuerpo)
  • DELETE: Sirve para eliminar datos (Generalmente solo recibe un parametro que es el destino de la peticion)
Nota importante: todos los metodos tienen un parametro adicional que es opcional para mandar propiedades adicionales (opciones)
