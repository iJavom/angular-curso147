Directivas

Es la herramienta que nos da angular para poder dar esos comportamientos adicionales sobre nuestro elementos en el DOM (Document Object Model)


Tipos de directivas:
  • Components Directives: Son las directivas mas comunes y son las que genralmente definen la estructura, estilo y logica
  • Structural Directives: Sirven para definir que se va a renderizar en neustro dom y por ende puede cambiar el diseño. Un ejemplo puede ser : *ngIf
  • Attribute Directives: Sirven para cambiar los atributos de nuestro elemento

Structural Directives


Son el conjunto de directivas que permiten definir la estructura de nuestro DOM / HTML a partir de las propiedades que presentamos en nuestro component

*ngIf


Sirve para determinar si un elemento de nuestro html se va a renderizar o no


Su sintaxis es en forma de atributo de una etiqueta y se pone *ngIf="Booleano"


En angular 17 es en el nuevo angular 17 es {{'@if(booleano){...}@else{...}'}}

*ngFor


Es una directiva para iterar sobre una coleccion o array y renderizar cada elemento de este


Su sintaxis es *nfor="let elemento of array; let i=index"

Ejemplo:

  • {{numero}} - esta es la posicion : {{i}}

ngSwitch


Permite renderizar o no elementos en nuestro html dependiendo de una expresion


La sintaxis consta de 3 directivas:

  • [ngSwitch]="expresion" -> Corresponde a la expresion a evaluar
  • *ngSwitchCase="valor" -> Corresponde a las posibles opciones
  • *ngSwitchDefault -> Corresponde en caso de no conseguir la opcion
